An oracle data contains the data sent by an oracle

```graphql
type OracleData {
  pubKeys: [String!]
  data: [Property!]
  matchedSpecIds: [ID!]
  broadcastAt: String!
}
```

#### [<code style={{ fontWeight: 'normal' }}>OracleData.<b>matchedSpecIds</b></code>](#)<Bullet />[`[ID!]`](/docs/testnet/graphql/scalars/id) <Badge class="secondary" text="list"/> <Badge class="secondary" text="scalar"/>

#### [<code style={{ fontWeight: 'normal' }}>OracleData.<b>broadcastAt</b></code>](#)<Bullet />[`String!`](/docs/testnet/graphql/scalars/string) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>

with a matching oracle spec.
It has no value when the oracle date does not match any oracle spec.
