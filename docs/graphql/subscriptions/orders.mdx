Subscribe to orders updates

```graphql
orders(
  marketId: ID
  partyId: ID
): [OrderUpdate!]
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>orders.<b>marketId</b></code>](#)<Bullet />[`ID`](/docs/testnet/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> ID of the market from which you want orders updates
> 

#### [<code style={{ fontWeight: 'normal' }}>orders.<b>partyId</b></code>](#)<Bullet />[`ID`](/docs/testnet/graphql/scalars/id) <Badge class="secondary" text="scalar"/>
> ID of the party from which you want orders updates
> 

### Type

#### [`OrderUpdate`](/docs/testnet/graphql/objects/order-update) <Badge class="secondary" text="object"/>
> An order update in Vega, if active it will be on the order book for the market
>
