Subscribe to price level market depth updates

```graphql
marketsDepthUpdate(
  marketIds: [ID!]!
): [ObservableMarketDepthUpdate!]!
```


### Arguments

#### [<code style={{ fontWeight: 'normal' }}>marketsDepthUpdate.<b>marketIds</b></code>](#)<Bullet />[`[ID!]!`](/testnet/api/graphql/scalars/id) <Badge class="secondary" text="non-null"/> <Badge class="secondary" text="scalar"/>
> ID of the market you want to receive market depth price level updates for
> 

### Type

#### [`ObservableMarketDepthUpdate`](/testnet/api/graphql/objects/observable-market-depth-update) <Badge class="secondary" text="object"/>
> Market Depth Update is a delta to the current market depth which can be used to update the
market depth structure to keep it correct
>
