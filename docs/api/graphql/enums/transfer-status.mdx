All the states a transfer can transition between

```graphql
enum TransferStatus {
  STATUS_PENDING
  STATUS_DONE
  STATUS_REJECTED
  STATUS_STOPPED
  STATUS_CANCELLED
}
```


### Values

#### [<code style={{ fontWeight: 'normal' }}>TransferStatus.<b>STATUS_PENDING</b></code>](#) 
> Indicates a transfer still being processed
> 

#### [<code style={{ fontWeight: 'normal' }}>TransferStatus.<b>STATUS_DONE</b></code>](#) 
> Indicates a transfer accepted by the Vega network
> 

#### [<code style={{ fontWeight: 'normal' }}>TransferStatus.<b>STATUS_REJECTED</b></code>](#) 
> Indicates a transfer rejected by the Vega network
> 

#### [<code style={{ fontWeight: 'normal' }}>TransferStatus.<b>STATUS_STOPPED</b></code>](#) 
> Indicates a transfer stopped by the Vega network
e.g: no funds left to cover the transfer
> 

#### [<code style={{ fontWeight: 'normal' }}>TransferStatus.<b>STATUS_CANCELLED</b></code>](#) 
> Indication of a transfer cancelled by the user
>
