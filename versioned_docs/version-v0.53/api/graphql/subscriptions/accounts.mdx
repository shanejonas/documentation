Subscribe to the accounts updates

```graphql
accounts(
  marketId: ID
  partyId: ID
  asset: String
  type: AccountType
): Account!
```


### Arguments

#### [`marketId`](#) ([`ID`](/mainnet/api/graphql/scalars/id))

ID of the market from which we want accounts updates


#### [`partyId`](#) ([`ID`](/mainnet/api/graphql/scalars/id))

ID of the party from which we want accounts updates


#### [`asset`](#) ([`String`](/mainnet/api/graphql/scalars/string))

Asset code


#### [`type`](#) ([`AccountType`](/mainnet/api/graphql/enums/account-type))

Type of the account


### Type

#### [`Account`](/mainnet/api/graphql/objects/account)

An account record
