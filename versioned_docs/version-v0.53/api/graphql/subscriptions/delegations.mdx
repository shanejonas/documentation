Subscribe to delegation data

```graphql
delegations(
  party: ID
  nodeID: ID
): Delegation!
```


### Arguments

#### [`party`](#) ([`ID`](/mainnet/api/graphql/scalars/id))

the party to subscribe for, empty if all


#### [`nodeID`](#) ([`ID`](/mainnet/api/graphql/scalars/id))

the node to subscribe for, empty if all


### Type

#### [`Delegation`](/mainnet/api/graphql/objects/delegation)
