Subscribe to the margin changes

```graphql
margins(
  partyId: ID!
  marketId: ID
): MarginLevels!
```


### Arguments

#### [`partyId`](#) ([`ID!`](/mainnet/api/graphql/scalars/id))

id of the party we want to subscribe for margin updates


#### [`marketId`](#) ([`ID`](/mainnet/api/graphql/scalars/id))

market we want to listen to margin updates (nil if we want updates for all markets)


### Type

#### [`MarginLevels`](/mainnet/api/graphql/objects/margin-levels)

Margins for a given a party
