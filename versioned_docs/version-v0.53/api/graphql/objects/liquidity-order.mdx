A special order type for liquidity providers

```graphql
type LiquidityOrder {
  reference: PeggedReference!
  proportion: Int!
  offset: String!
}
```


### Fields

#### [`reference`](#) ([`PeggedReference!`](/mainnet/api/graphql/enums/pegged-reference))

The value to which this order is tied


#### [`proportion`](#) ([`Int!`](/mainnet/api/graphql/scalars/int))

The proportion of the commitment allocated to this order


#### [`offset`](#) ([`String!`](/mainnet/api/graphql/scalars/string))

Offset from the pegged reference
