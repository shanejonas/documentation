An order in the VEGA network found by referenceID

```graphql
orderByReference(
  reference: String!
): Order!
```


### Arguments

#### [`reference`](#) ([`String!`](/mainnet/api/graphql/scalars/string))

Reference for an order


### Type

#### [`Order`](/mainnet/api/graphql/objects/order)

An order in Vega, if active it will be on the OrderBook for the market
