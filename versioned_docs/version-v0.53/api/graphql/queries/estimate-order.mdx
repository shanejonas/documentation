return an estimation of the potential cost for a new order

```graphql
estimateOrder(
  marketId: ID!
  partyId: ID!
  price: String
  size: String!
  side: Side!
  timeInForce: OrderTimeInForce!
  expiration: String
  type: OrderType!
): OrderEstimate!
```


### Arguments

#### [`marketId`](#) ([`ID!`](/mainnet/api/graphql/scalars/id))

ID of the market to place the order


#### [`partyId`](#) ([`ID!`](/mainnet/api/graphql/scalars/id))

ID of the party placing the order


#### [`price`](#) ([`String`](/mainnet/api/graphql/scalars/string))

Price of the asset


#### [`size`](#) ([`String!`](/mainnet/api/graphql/scalars/string))

Size of the order


#### [`side`](#) ([`Side!`](/mainnet/api/graphql/enums/side))

Side of the order (Buy or Sell)


#### [`timeInForce`](#) ([`OrderTimeInForce!`](/mainnet/api/graphql/enums/order-time-in-force))

TimeInForce of the order


#### [`expiration`](#) ([`String`](/mainnet/api/graphql/scalars/string))

expiration of the the order


#### [`type`](#) ([`OrderType!`](/mainnet/api/graphql/enums/order-type))

type of the order


### Type

#### [`OrderEstimate`](/mainnet/api/graphql/objects/order-estimate)

An estimate of the fee to be paid by the order
