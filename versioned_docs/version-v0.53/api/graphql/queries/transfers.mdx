get a list of all transfers for a pubkey

```graphql
transfers(
  pubkey: String!
  isFrom: Boolean
  isTo: Boolean
): [Transfer!]
```


### Arguments

#### [`pubkey`](#) ([`String!`](/mainnet/api/graphql/scalars/string))

the pubkey to look for


#### [`isFrom`](#) ([`Boolean`](/mainnet/api/graphql/scalars/boolean))

is the pubkey on the sending part of the transfer


#### [`isTo`](#) ([`Boolean`](/mainnet/api/graphql/scalars/boolean))

is the pubkey in the receiving part of the transfer


### Type

#### [`Transfer`](/mainnet/api/graphql/objects/transfer)

A user initiated transfer
